# Installion

## Clone repository

```Bash
git clone https://github.com/AppBlitz/github-user-activity.git
cd github-user-activity
```

## Build project  with use cargo

```bash
cargo build --release
```

## Run executable 
```Bash
./target/release/github-user-activity
```
